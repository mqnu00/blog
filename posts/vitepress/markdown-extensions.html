VitePress中的Markdown扩展功能 ​

在使用VitePress搭建博客时，了解其内置的Markdown扩展功能是非常有帮助的。本文将详细介绍一些常用的扩展功能。

语法高亮 ​

VitePress提供由 Shiki 驱动的语法高亮，还具有行高亮等附加功能：

示例代码

md
```js{4}
export default {
  data () {
    return {
      msg: 'Highlighted!'
    }
  }
}
```

Output

js
export default {
  data () {
    return {
      msg: 'Highlighted!'
    }
  }
}
